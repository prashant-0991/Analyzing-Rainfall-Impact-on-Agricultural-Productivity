# Analyzing Rainfall Impact on Agricultural Productivity

## 📌 Project Overview
This project features an interactive Power BI dashboard titled **"Production Analysis Based on Rainfall Patterns"**. The primary objective is to evaluate how precipitation variations influence agricultural yield across different states and crop types. By combining environmental and production data, this dashboard provides actionable insights into climate-resilient farming, resource optimization, and regional agricultural performance.

## 📊 Key Features & KPIs
The interactive report delivers deep insights through several key analytical views:

* **State-Wise Productivity Analysis:** A regional breakdown mapping crop production and yield to identify high-performing agricultural hubs and areas highly vulnerable to rainfall deficits.
* **Top 5 Rainfall - Productivity Relationship:** A focused analysis on the top 5 regions/crops where rainfall patterns have the most direct and significant correlation with total productivity.
* **Top 10 Rainfall to Crop Yield Analysis:** A deep-dive matrix highlighting the top 10 crops, showcasing how specific rainfall thresholds optimize or hinder their specific yield metrics.
* **Rainfall Trends Analysis Over Time:** A historical time-series visualization tracking precipitation shifts across seasons and years to uncover long-term climate trends and predictability.

## 🛠️ Tech Stack & Skills Demonstrated
* **Data Visualization:** Power BI Desktop
* **Data Transformation:** Power Query (Data cleaning, column splitting, handling missing values, and data type formatting)
* **Data Modeling:** Established star schema relationships with proper cardinality (One-to-Many) between dimension and fact tables.
* **DAX (Data Analysis Expressions):** Implemented advanced DAX measures utilizing functions like `CALCULATE`, `SUMX`, and time-intelligence functions for dynamic calculations.
